# Automatic-Login-Tool
Automated login script built with Selenium that fills credentials, handles page loading, and signs in automatically. Useful for learning browser automation, testing login flows, and reducing repetitive manual logins.

A simple Python project that automates logging into a website using Selenium.
## Features
- Opens a browser automatically
- Navigates to the login page
- Enters username and password
- Submits the login form
- Demonstrates basic Selenium automation
## Tech Used
- Python
- Selenium WebDriver
- ChromeDriver / WebDriver Manager
